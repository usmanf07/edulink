# Edulink a MERN Stack Project for Institutes & Students
- What is Edulink?
Edulink is a MERN stack based dynamic platform that connects students and institutes from all corners of the world. It offers a wide range of functionalities to help students prepare for entry tests and search for and register in institutes worldwide. üåç

## Installation
  1. Clone the repository to your local machine.
  2. Install the dependencies for both the server and client.
     ```
       cd edulink
      npm install

      cd client
      npm install
     ```
  3. Run the command npm start
  4. Also run the backend folder using `npm run dev`
